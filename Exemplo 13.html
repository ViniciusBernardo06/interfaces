Exemplo 13 HTML - Formulários

Os formulários HTML são uma principais formas de se interagir com aplicações web. Esses formulários utilizam diversos componentes que melhorarão a experiência do usuário.

Para trabalhar com formulários devemos usar a tag

em conjunto comoutras tags que vão representar os componentes de formulários, tais como: labels(rotúlos), inputs(caixa de texto), botões, elementos de checagem, de seleção de ntre outros componentes. Neste exemplo, demonstraremos o uso dos métodos Get e Post

Métodos do Protocolo HTTP

Get

Quando trabalhamoscom a criação de formulários html, nós deveremos na parte das vezes utilizar o protocolo de rede http, esse protocolo possui vários métodos que são utilizados para facilitar o processo de comunicação entre o cliente e o servidor.

O método GET é aquele que anexa os dados do formulário à uma url, esses dados do formulários são vísíveis na barra de endereço do navegador.

É importante saber que este método não recomendado para operções de inclusão, pois os dados vão aprecer para qualquer pessoa,o que pode ser inseguro.

Post

Já o método POST é aquele que envia os dados para o servidor no corpo da página. Essa informações não são vísives par o cliente sendo um pouco mais seguras que quando utilizamos o método GET.
