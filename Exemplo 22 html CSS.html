Cores estilos de Fundo

Uso do background com sistema de cores RGB e RGBA

Sistema de cores RGB

O sistema de cores RGB representa um esquema de cores baseado nas cores R(red)G(green) e o B(blue), cada parâmetro do rgb define a intensidade da cor e ela varia de 0 255. Por exemplo rgb(255,99,71)

Sistema de cores RGBA

o sistema de cores rgba são uma extensão dos valores rgb, com um canal alfa (a), que especifica a opacidade que pode variar de 0 a 1, Sendo o 0 totalmente transparente e o 1 totalmente opaco. Por exemplo rgba(35,99,71,0.3)
