Visao Geral sobre o CSS

CSS é um acrônimo para folhas de estilo em cascata. Essas técnica visa personalizar, estilizar um documento HTML. Para isso, usamos regras CSS, que representam uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • SeletorIndica quais elementos serão afetados pelo estilo definido.
  • Bloco de declaração É o bloco composto por propriedades e valores, esse bloco será aplicado ao elemento html.

Tipos de seletores

Em CSS seletores são padrões usados para selecionar e estilizar os elementos HTML em uma página web. Eles definem quais elementos receberção as regras de estilos que serão declaradas dentro das chaves. Possuímos vários tipos de seletores, os principais são:

Seletores de ID

O seletor de ID é aquele que aplica regra css a um componente html que tenha um atributo id no componente html. Este seletor de id deve ser aplicado a um unico elemento html, haja vista que o id deve ser único. Este seletor tem prioridade sobre os demais seletores(de classse e de tipo)

Este seletor sempre começara com um "#", seguido nome do seletor e suas propriedades e valores .

Neste Exemplo usamos seletores de id,de classe e tipo para estilizar o documento html.
