Exemplo 8 - Uso das tags de imagem <img>, video e iframe

Este exemplo mostra diferentes formas de usar as tags img, video e iframe, incluindo o uso do figure e figcaption para agrupar midias.

As tags abordadas são :

  • <img>: Tag usada para inserir imagens na pagina. Essa tag é utilizada em conjunto com os atributos src, alt e width e height.
  • <videop>: Usada para inserir videos na pagina
  • <iframe>
    • Tag usada para incorporar conteúdo externo como por exemplo vídeos do youtube
