Exemplo 4 HTML - Estrutura geral de um documento HTML

Este exemplo mostra a estrutura minima de um documento html. Aqui estao as principais tags utilizadas:

  • <!DOCTYPE html >Define o tipo de documento HTML, ou seja a versao do html que sera usada na pagina
  • <html >A tag html é o elemento principal do documento html, ela define o container raiz de toda página.
  • <head >A tag head contém os metadados que serão interpretados pelo navegador. Nela encontraremos o título da página e links e arquivos para configuração do documento.
  • <body >: A tag body e responsavel pela exibição do conteúdo visivel do documento, ou seja, tudo que estiver neste bloco será visivel para este usuario
